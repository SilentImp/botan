Using Ramjet

Install it with npm install ramjet, or download ramjet.js. Head over to GitHub for more detailed instructions.

More info

Ramjet is free and open source software released under the MIT License. The code is available on Github. Issues and pull requests welcome!

It has been successfully tested in IE9+, Chrome (desktop and Android), Firefox, Safari 6+ and mobile Safari - please raise an issue on GitHub if your experience differs!
